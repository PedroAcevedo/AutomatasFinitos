# AutomatasFinitos
Compiler assignment

Gloria al bravo pueblo
que el yugo lanzó
la Ley respetando
la virtud y honor.

Abajo cadenas
gritaba el señor
y el pobre en su choza
libertad pidió
a este santo nombre
tembló de pavor
el vil egoísmo
que otra vez triunfó.

Gritemos con brío
Muera la opresión
Compatriotas fieles,
la fuerza es la unión;
y desde el Empíreo
el Supremo Autor,
un sublime aliento
al pueblo infundió.

Unida con lazos
que el cielo formó
la América toda
existe en nación
y si el despotismo
levanta la voz,
seguid el ejemplo
que Caracas dió.
